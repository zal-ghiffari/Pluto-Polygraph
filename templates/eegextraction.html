EEG Data Extraction

Pluto polygraph bekerja dengan mendeteksi berdasarkan nilai pada gelombak otak ketika proses wawancara. Jadi sebelumnya yuk, simak cara ekstraksi datanya sebelum di analisis.

Guide To Extraction

Prepare Tools


1. Menggunakan Alat EEG Headset, pada platform ini disarankan menggunakan alat dari Neurosky Mindwave
2. Download dan install aplikasi bawaan EEG Headset untuk koneksi EEG Headset
3. Pastikan baterai perangkat cukup dan daya tersalurkan dengan baik


Prepare For Code


1. Install Python sesuai versi yang disukai (Seharusnya sudah install python jika webnya sudah berjalan)
2. Install library dengan perintah : 

                  pip install pandas
                  pip install numpy

3. Buat file python baru kemudian ketik code berikut ini, jika tidak ingin mengetik code dapat mendownload dengan button dibawah

                  tn=Telnet('127.0.0.1',13854)

                  signalLevel=0

                  while time.time() - start < 30:
                      blinkStrength=0

4. Simpan file ke direktori masing-masing


Data Extraction Proccess


1. Pastikan alat EEG Headset sudah terkoneksikan dengan baik
2. Bukan Comand Promt atau CMD, dan arahkan direktori menuju folder tempat menyimpan code sebelumnya
3. Jalankan code program dengan perintah : 

                  python <namafile.py> <hasilputput.csv>

4. Lakukan interview atau introgasi selama script berjalan
5. Setelah itu, data akan muncul dalam bentuk CSV pad direktori anda dan data siap untuk dianalisis
